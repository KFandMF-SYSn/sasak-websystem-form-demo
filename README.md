# websystem-form-demo

企画対応システムと即売会とかのシステムは、身近だから作ってみたら楽しそうだなって思う、って思って作ってみた小さなデモアプリ

文言とかバリデーションの情報をもったJSONとかを中間表現で持って、
- 中間表現をポチポチしたら生成できるシステム
- 中間表現からフロントの画面を生成できるコンポーネント
- 中間表現と、POSTされたデータを渡せばバリデーションしてくれる関数（バックで利用）

をつくりたいのうち、「中間表現からフロントの画面を生成できるコンポーネント」のイメージを実装してみた
`App.tsx`にある`FormBuilder`がそれ。
